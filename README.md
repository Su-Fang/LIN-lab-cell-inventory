🚀 **Lin Lab Cell Inventory Management System** 📗📕📙

這是一個專為實驗室開發的智慧化液態氮庫存管理系統。 針對傳統 Excel 紀錄難以與實體位置對應的痛點，本系統提供直觀的 5x5 空間網格視覺化與快速檢索功能，大幅提升細胞找尋與存放的效率。

🚀 **核心功能 (Key Features)**

  🔎 智慧快速檢索–全域搜尋：支援模糊搜尋，輸入細胞名稱即可跨 Rack/Box 鎖定位置。

  🌎 即時定位：直接顯示目標細胞的鐵架編號、盒子編號與座標。

  🧊 實體模擬：完美呈現 5x5 冷凍盒配置，綠色代表已佔用，灰色代表空位。智慧處理長字串，確保在手機或電腦上查看時網格皆整齊不變形。

  📊 庫存動態儀表板：即時計算液態氮桶細胞的使用率與剩餘空間並自動推薦空位最多的盒子，導引使用者進行空間優化。

  🖨️ 專業標籤列印模式：專為列印設計的 CSS，將文字強制轉為純黑色，確保標籤易於讀取。 
  
  🧟‍♂️ 隱身導航：列印時自動隱藏操作按鈕，僅保留核心網格。

🛠️ **技術細節 (Tech Stack)**

  Frontend: Streamlit (Python-based web framework) 

  Data Source: Google Sheets API (CSV export) 

  Styling: Custom CSS injection for print optimization & UI enhancement

🚀 **Deployment: Streamlit Cloud**
